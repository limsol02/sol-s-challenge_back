### 1. 흥미로운 도입
상상해보세요. 당신은 마법사의 비밀 연구소에 들어섰습니다. 이곳에서 마법사는 다양한 마법의 
주문을 사용해 눈부신 효과를 만들어냅니다. 이제 JSP, JSTL, 그리고EL은 마법사의 도구와 같습니다.
마법사가 복잡한 주문을 쉽고 효과적으로 외우는 방법을 찾았듯, 이 기술들은 복잡한 웹 프로그래밍 작업을
단순하고 강력하게 만들어줍니다. JSP는 마법사의 주문서, JSTL은 주문을 간결하게 만드는 마법이 주문 모음집,
그리고 EL은 마법의 힘을 직접적으로 간단하게 끌어내는 방법입니다.이제 우리는 이 마법의 세계로 들어가 마법사의 주문을 배우고,
 어떻게 멋집 웹 페이지를 마법처럼 만들어 낼 수 있는 탐험해보겠습니다.

### 2. 학습목표
- JSP, JSTL, EL의 기본 개념과 작동원리 이해
- JSTL 태그 라이브러리와 EL문법의 사용법 숙지
- JSP에서 JSTL과 EL을 사용하여 동적 웹페이지 생성 방법 학습
- 코드의 재사용성과 유지보수의 용이성 향상

### 3. 주요 핵심키워드와 개념
- **JSP(Jakarta Server Pages)** : HTML내에 자바 코드를 삽입하여 동적인 웹 페이지를 생성하는 서버 사이드 스크립팅 기술
- **JSTL(Jakarta Standard Tag Library)** : JSP 페이지 내에서 자바 코드 없이 태그를 사용하여, 
	반복, 조건문, 국제화 등의 작업을 수행하는 표준 태그 라이브러리
- **EL(Expression Language)** : JSP 페이지 내에서 복잡한 자바 코드를 간단한 표현식으로
	작성할 수 있게 하는 언어

### 4. 생각하고 토론해볼 내용
- JSP와 일반 HTML의 차이점은 무엇이며, 어떤 상황에서 JSP를 사용하는 것이 더 적합한가? 
- JSTL과 전톡적인 자바 스크립트릿(seriptlet) 방식 사이의 장단점은 무엇인가?
- EL을 사용함으러써 얻는 이점과 JSP에서의 데이터 접근 방식에 대한 토론.

### 5. 간단한 퀴즈m
1. JSTL의 `c:forEach` 태그는 어떤 기능을 수행하나요?
2. EL에서 `${}`와 `#{}`의 차이점은 무엇인가요?
3. JSP 페이지에서 선언된 자바 변수를 EL을 사용하여 어떻게 접근할 수 있나요?

### 6. 싦무적용예시
- **사용자 목록 표시** : 데이터베이스에서 사용자 목록을 가져와 JSP 페이지에 테이블 형식으로 표시.
	`c:forEach` 를 사용하여 각 사용자 데이터를 행으로 표시
- **로그인 상태 표시** : 로그인 여부에 따라 다른 메뉴 옵션을 보여주는 기능.
	`c:choose`, `c:when`, `c:otherwise` 태그를 사용하여 조건에 따라 다른 HTML코드를 표시
- **에러 메시지 처리** : 폼 제출 과정에서 발생한 에러 메세지를 사용자에게 보여주는 기능.
	EL을 사용하여 서버 측에서 설정한 에러 메세지를 JSP 페이지에 표시
